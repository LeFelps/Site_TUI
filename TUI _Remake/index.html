Nossa

História

A TUI foi criada em 2017 com o objetivo de desenvolver projetos pautados pelos idealizadores, como aplicativos móveis e jogos

Somos uma empresa de tecnologia focada em desenvolvimento de sistemas especializados, expandindo para a integração de novas tecnologias e diferentes serviços.

A pesar de inicialmente desenvolvermos aplicações especializadas somente para desktop, a perspectiva é um maior investimento em aplicações WEB e mobile, trazendo uma maior versatilidade e usabilidade

Nossos serviços de Software

Software

Sob Medida

A TUI oferece um serviço de desenvolvimento de sistemas sob demanda, seguindo um processo organizacional. Nossa equipe de Analistas será enviada para fazer o levantamento de requisitos na sua empresa, e ao entender como funciona o seu negócio, faremos um sistema feito sob medida para você, visando maximizar sua performance e resolver seus problemas administrativos

Software

Licensiado

Softwares como o TUI-Gerenciamento foram desenvolvidos para tornar mais acessível o uso de aplicações de gerenciamento empresarial. A partir de uma mensalidade, conseguimos fornecer um software mais barato que atende suas necessidades gerais, úteis para a maioria dos diferentes tipos de negócio.

Aplicações

Web

Com desenvolvimento de sites, ou aplicações web, temos como finalidade fornecer uma ferramenta que permita a divulgação e expansão do seu negócio e das informações de sua empresa, para que seus clientes tenham fácil acesso a qualquer momento. Os sites são totalmente personalizados, atendendo às suas necessidades e preferências, em conjunto com a elaboração de ideias da nossa equipe de design

Nossos serviços de Design

Design

de Logotipo

Elaboração de logotipo com estilo simplista e moderno, para empresas de qualquer segmento. Criamos o logotipo de acordo com com direito a 3 revisões. Você nos manda instruções de como quer que seja a logo e nós desenvolvemos

Design

de Cartão

Fazemos cartões de visita personalizados de acordo com o seu negócio. Você pode escolher os acabamentos e formato que mais lhe agradar, com direito a algumas revisões

É necessário possuir ou contratar o desenvolvimento de uma LOGO para que o cartão seja feito
